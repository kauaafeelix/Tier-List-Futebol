### Descrição da Atividade:

Nesta atividade, você irá criar uma galeria de imagens interativa usando JavaScript para aplicar conceitos de seleção e manipulação de elementos do DOM e eventos de drag and drop. O objetivo é permitir que os usuários arrastem e soltem imagens entre duas seções: a Galeria Principal e a Seção Favoritos.

Você deverá implementar uma funcionalidade em que as imagens podem ser arrastadas para a seção de favoritos, recebendo uma borda de destaque ao serem soltas nessa área. Além disso, cada imagem na seção de favoritos deve poder ser removida ao ser clicada.

### Objetivos de Aprendizagem:

Praticar seleção e manipulação de elementos no DOM.

Usar eventos para manipular e alterar elementos.

Implementar e personalizar a funcionalidade de drag and drop em JavaScript.

Manipular classes e estilos para melhorar a experiência do usuário.

### Instruções:

*Criação da Estrutura HTML*: Crie duas seções: uma Galeria Principal com imagens e uma Seção Favoritos onde as imagens poderão ser soltas
.
Cada imagem na Galeria Principal deve ter a classe draggable e o atributo draggable="true".

*Estilos CSS* : Estilize as duas seções para que visualmente representem uma galeria e uma área de favoritos.
Aplique um estilo visual para a área de drop (dropzone) quando uma imagem está sendo arrastada sobre ela.

*JavaScript*: Implementação do Drag and Drop:Selecione todas as imagens na galeria e implemente os eventos para manipular o arrasto.

No evento drop, verifique se a imagem já existe nos favoritos antes de adicioná-la.

Implemente uma função para remover uma imagem dos favoritos ao ser clicada.

### Desafios Extras (Opcional):
Permita que as imagens possam ser arrastadas da área de favoritos de volta para a galeria principal.
Adicione uma contagem ao lado da seção de favoritos para exibir o número de imagens favoritas.
Permita que as imagens na área de favoritos possam ser reorganizadas por meio de drag and drop.
Entrega: Faça o upload dos arquivos HTML, CSS e JavaScript para a atividade no Google Classroom.
